# Meetup Agent-Skills 2026

## Agent Skills - von Basics bis Highlevel Workflows

Angewandte KI. Hands-on. Offen für alle.

---

## Worum geht's?

Dieses Repository enthaelt das komplette Workshop-Material fuer einen Meetup-Track
zu **Agent Skills** mit OpenCode.

Ziel ist ein praxisnaher Einstieg: von Grundlagen ueber Skill-Nutzung bis hin zu
fortgeschrittenen Szenarien mit eigenen Skill-Strukturen und Coding-Bezug.

**Wir fokussieren auf:**

- Verstehen, was Skills sind und wann sie sinnvoll sind
- Vergleiche **ohne Skill** vs. **mit Skill**
- Sichere Nutzung externer Skills (z. B. ueber `skills.sh`)
- Eigene Skill-Ideen schnell umsetzen und iterieren

---

## Format

Nach einem kurzen Einstieg arbeiten Teilnehmende Schritt für Schritt durch die
vorbereiteten Materialien - allein oder im Team.

Der Schwerpunkt liegt auf:

- direkt nutzbaren Prompts,
- praktischen Skill-Tests,
- Vergleich und Reflexion (Qualität, Konsistenz, Aufwand),
- Transfer auf eigene Use Cases.
- Community-Austausch: Wer nutzt Skills? Wofür? Wie sind die Erfahrungen?

---

## Fokusthema: Agent Skills in der Praxis

Wir behandeln Agent Skills als wiederverwendbare Arbeitsanweisungen für
LLM-Agenten - inklusive Struktur/Template, Triggern, Guardrails und Validierung.

### Mögliche Beispiele

- Dokumente strukturieren (Briefing, Summary, Vorlagen)
- Office-Workflows (Excel, PowerPoint)
- Skill-gestützte Coding-Abläufe
- Eigene Mini-Skills für wiederkehrende Aufgaben

---

## Ablauf im Workshop

| Phase | Fokus |
|------|-------|
| Setup | OpenCode Einstieg und Arbeitsweise |
| Intro | Was sind Skills, wann nutzen, Risiken und Grenzen |
| Basics | Ersten Skill bauen und testen |
| Midlevel | Office-Aufgaben mit/ohne Skill vergleichen |
| Highlevel | MCP-Builder und eigene Skill-Struktur mit Code |
